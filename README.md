# GhostPings [WIP]
A Minecraft Fabric mod for pinging locations to other players  
Currently under development  

## Showcase
[Watch 9 second demo (Vimeo)](https://vimeo.com/1144544921?autoplay=1&loop=1)  

## Features
- Multiplayer support  
- Colour customization  
- Channels  
- Ping spam prevention  
- In-world 3D rendering  
- HUD rendering of player skin icons  

## How to use
The default keybind to create new pings is C, but this can be changed in the settings.  
The active channel can be changed with the command `/ghostpings channel <channel_name>` [WIP]  
Ping colours can be customized via `/ghostpings primary <rgb_hex>` and `/ghostpings secondary <rgb_hex>` [WIP]  
- For example: `/ghostpings primary #FFB500` and `/ghostpings secondary #F00` for the same colours as in the showcase.  

## How to install
- Download the latest version from the releases tab [WIP]  
- Simply place the jar file in the mod folder  
(Requires Fabric)
